How This Dashboard Works

The dashboard edits the source files that power the songbook: the song list, the song map, and each song text file. It can save drafts in this browser, export backups, and in supported browsers write the updated files directly into your local project folder.

Because the live site is static, browser edits do not publish automatically to GitHub Pages. After writing changes into the local project, commit and push to make them live.
